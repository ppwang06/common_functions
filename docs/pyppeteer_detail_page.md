###### pyppeteer说明

```shell script
headless: False以无头浏览器的形式运行
dumpio: 是否将浏览器进程stdout和stderr传递到process.stdout和中process.stderr.默认为False。
handleSIGINT: (bool)在Ctrl + C上关闭浏览器进程,默认为 True
handleSIGTERM: (bool)关闭SIGTERM上的浏览器进程,默认为True
handleSIGHUP: (bool)关闭SIGHUP上的浏览器进程,默认为 True
ignoreHTTPSErrors: (bool)是否忽略HTTPS错误,默认为 False

```
